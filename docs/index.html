Rakutore Anchor の主な特徴

シンプルなナンピン設計

複雑なフィルターや多重ロジックではなく、 「今のポジション数で、次のナンピンの目安」が一目で分かる設計です。 小さく始めて、慣れてから設定を調整できます。

見やすいパネル操作

Start BUY / Start SELL / Close All / MOVE / Reset など、 パネルからワンクリックで操作が可能です。 その日の獲得pipsもパネル上で確認できます。

サポートとアップデート

サポート窓口（support@rakutore.jp）にて、 設定に関するご相談や、口座変更の手続き方法などを受け付けています。 不具合修正などのアップデートもこのサイトからご案内予定です。
