シンプルなナンピン設計

複雑なフィルターや多重ロジックではなく、 「今のポジション数、ナンピンまでの目安」が一目で分かる設計です。 小さく始めて、慣れてから設定を調整できます。

見やすいパネル操作

Start SELL / Start BUY / Close All / MOVE / Reset など、 パネルからワンクリックで操作が可能です。 その日の獲得pipsもパネル上で確認できます。
